 

# Django + React Task Manager - Internshala Project

## Description:
Simple Task Manager project with Django backend and React frontend.

## Features:
- CRUD Tasks
- REST API using Django
- React frontend consuming API

## Setup Instructions:

### Backend:
1. cd backend
2. pip install -r requirements.txt
3. python manage.py migrate
4. python manage.py runserver

### Frontend:
1. cd frontend
2. npm install
3. npm start
